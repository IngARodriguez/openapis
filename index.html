openai.sh 
# Sin streaming
curl https://tapeless-recluse-disperser.ngrok-free.dev/openai/v1/chat/completions \
  -H "Authorization: Bearer admin" \
  -H "content-type: application/json" \
  -d '{
    "model": "gpt-5.5",
    "max_tokens": 1024,
    "messages": [{"role": "user", "content": "Hola"}]
  }'

  -H "Authorization: Bearer admin" \
  -H "content-type: application/json" \
  -d '{
    "model": "gpt-5.5",
    "max_tokens": 1024,
    "stream": true,
    "messages": [{"role": "user", "content": "Hola"}]
  }'

openai_example.py 
# Sin streaming
from openai import OpenAI

client = OpenAI(
    api_key="admin",
    base_url="https://tapeless-recluse-disperser.ngrok-free.dev/openai/v1",
)

response = client.chat.completions.create(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "Hola"}],
)
print(response.choices[0].message.content)

# Con streaming
with client.chat.completions.stream(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "Hola"}],
) as stream:
    for text in stream.text_stream:
        print(text, end="", flush=True)

anthropic_example.py 
# Sin streaming
from anthropic import Anthropic

client = Anthropic(
    api_key="admin",
    base_url="https://tapeless-recluse-disperser.ngrok-free.dev/anthropic",
)

response = client.messages.create(
    model="claude-opus-4-7",
    max_tokens=1024,
    messages=[{"role": "user", "content": "Hola"}],
)
print(response.content[0].text)

# Con streaming
with client.messages.stream(
    model="claude-opus-4-7",
    max_tokens=1024,
    messages=[{"role": "user", "content": "Hola"}],
) as stream:
    for text in stream.text_stream:
        print(text, end="", flush=True)
Características

Pensado para builders

Drop-in compatible. Sin reescribir tu stack.

Drop-in compatible

Cambia el base_url y la clave. Tu código existente con los SDKs oficiales de OpenAI y Anthropic sigue funcionando — cero refactor.

Streaming nativo

SSE end-to-end sin buffering ni proxies opacos. Los tokens llegan al instante, con la misma latencia que llamar directo al proveedor.

Autenticación simple

Una clave única para todos los proveedores. Olvídate de gestionar dashboards, facturas y rotaciones por separado.

Multi-proveedor

OpenAI y Anthropic bajo la misma URL. Cambia de modelo en una línea sin tocar el resto de tu stack.

Único pago

Sin suscripciones, sin renovaciones, sin sorpresas a fin de mes. Pagas una vez y usas. Punto.
